# HomeHero - Volume 1: Startup Foundation Prompt Book

This document compiles the master prompt engineering templates used to validation, model, and bootstrap the **HomeHero** startup foundation. 

---

## 1. Startup Idea Validation Prompts

### Prompt 1.1: Core Concept Testing
```
Act as a Startup Business Analyst and Hyperlocal Market Researcher. Analyze the viability of "HomeHero" - a hyperlocal, real-time MERN platform matching home service professionals (electricians, plumbers, carpenters) with users in a 15km radius. 

Deliver a comprehensive validation report answering:
1. Target Customer Profile (pain points, demographics).
2. Primary Value Proposition vs standard agencies.
3. Market size estimation models (TAM, SAM, SOM) for tier-1 cities in India.
4. Risk assessment matrix (safety, reliability, service quality).
```

## 2. Market Research Prompts

### Prompt 2.1: Industry Metrics Analysis
```
You are a Principal Market Researcher. Provide a data-driven market report on the global and regional On-Demand Home Services Market from 2024 to 2030.

Cover:
1. Compounded Annual Growth Rate (CAGR) calculations.
2. Market share distribution by category (Cleaning vs Repairs vs Maintenance).
3. Primary drivers (mobile penetration, urbanization, disposable income shifts).
4. Major market barriers and regulatory compliance constraints.
```

---

## 3. Competitor Analysis Prompts

### Prompt 3.1: Competitor Matrix Mapping
```
Act as a Competitive Intelligence Specialist. Create a detailed competitor matrix comparing HomeHero against established players (e.g., Urban Company, TaskRabbit, Angi).

Analyze:
1. Pricing models (commissions, flat fees, subscription options).
2. Service fulfillment workflow (immediate dispatch vs scheduled bidding).
3. Vetting and verification processes for service providers.
4. Core vulnerabilities or friction points in competitor platforms (reviews, payouts).
```

---

## 4. Customer Interviews Prompts

### Prompt 4.1: Interview Questionnaire & Persona Discovery
```
You are a UX Researcher and Customer Empathy Specialist. Design two interview scripts: one targeting homeowners (customers) and one targeting independent service professionals (technicians).

Ensure the questions uncover:
1. Trust triggers (background checks, reviews, ratings).
2. Friction points in finding work or hiring help.
3. Payment transparency and expectations.
4. App retention drivers (recurring bookings, reward loops).
```

## 6. Revenue Model Prompts

### Prompt 6.1: Financial Pricing & Commision Structures
```
Act as a Chief Financial Officer. Model three distinct revenue streams for HomeHero and run feasibility models:
1. Platform commission (defaulting to 15% per booking).
2. Dynamic Surge Pricing (surcharges for monsoon, night, holidays).
3. Premium Technician Tiers (paid visibility upgrades).
Provide projection calculations for monthly operational break-even limits.
```

## 7. MVP Planning Prompts

### Prompt 7.1: Scoping the Minimal Viable Product
```
You are a Principal Product Manager. Outline the MVP product scope for HomeHero. 

Define:
1. Must-Have features for launch (OTP-verification, booking matchmaker, Razorpay integration).
2. Excluded features for v1 (subscription clubs, team bookings).
3. Success metrics (Daily Active Users, booking completion rate, rating averages).
```

## 8. Feature Prioritization Prompts

### Prompt 8.1: MoSCoW Prioritization
```
Act as a Technical Product Owner. Apply the MoSCoW prioritization framework to the HomeHero backlog.

Categorize features into:
- Must-Have (e.g., Auth, booking checkout, tracking).
- Should-Have (e.g., custom addressing, photo uploads).
- Could-Have (e.g., technician routing ETA, messaging).
- Won't-Have (v1) (e.g., multi-city expansion).
```
