# Razorpay Integration

## Overview
This document describes the integration of **Razorpay** payment gateway into HomeHero.

- Obtaining API keys (Key ID and Secret)
- Server‑side order creation
- Client‑side checkout flow using Razorpay Checkout
- Webhook handling for payment success/failure
- PCI‑DSS considerations and tokenisation

*Include code examples for Node/Express backend and React checkout component.*
