# HomeHero - Logo Usage & Guidelines

This document details the usage guidelines, spacing rules, color restrictions, and lockups for the HomeHero logo.

---

## 1. Logo Visual Lockup Options

The HomeHero logo consists of a **Hero Shield** icon combined with clean, modern text:

```
    [ ]   <-- Shield Symbol (Solid Indigo, #6366f1)
  [     ]
   [   ]
    [ ]
  HomeHero  <-- Outfit Font Wordmark (Pure White, #ffffff)
```

- **Vertical Lockup (Default)**: Shield icon centered above the "HomeHero" wordmark. Used for mobile app loading screens and splash pages.
- **Horizontal Lockup**: Shield icon placed to the left of the "HomeHero" wordmark. Used for navigation bars and dashboard headers.

---

## 2. Safe Spacing & Padding Rules

To maintain visual impact, always keep a minimum clear space around the logo equivalent to the height of the shield symbol:

```
┌──────────────────────────────────────────────┐
│                  Clear Space                 │
│    ┌────────────────────────────────────┐    │
│    │             [Shield]               │    │
│    │             HomeHero               │    │
│    └────────────────────────────────────┘    │
│                  Clear Space                 │
└──────────────────────────────────────────────┘
```

- **Minimum Width**: The logo wordmark should never be rendered smaller than **120 pixels** on screen or **25 millimeters** in print.
- **Padding**: Enforce a minimum **16px (1rem)** padding buffer between the logo and other navigation menu items.

---

## 3. Prohibited Logo Alterations

*   Do not rotate, stretch, or skew the logo shield or wordmark.
*   Do not change the logo colors (must remain solid Indigo and White, or Slate on light backdrops).
*   Do not add drop shadows or gradients to the shield symbol.
