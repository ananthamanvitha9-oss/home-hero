# HomeHero - Logo Usage & Brand Identity Guidelines

**Prepared by**: Brand Identity Designer  
**Target Audience**: UI Engineers, Graphic Designers, & Marketing Partners  
**Focus**: Logo geometries, clearance specifications, color variants, and app icon rules

---

## 1. Logo Concept & Meaning

The HomeHero logo is a unified brand symbol combining safety, home, and reliability:

*   **The Shield**: Represents safety, security, and our verified vetting process.
*   **The Home Silhouette**: Formed inside the shield geometry, representing the domestic services sector.
*   **Brand Meaning**: A shield protecting the home, symbolizing the trust and safety HomeHero brings to residential repairs.

---

## 2. Logo Configurations & Variations

### 2.1 Primary Logo (Vertical Stack)
Used for mobile splash screens, landing page hero modules, and main marketing materials.

```
      [ Shield Symbol ]
    ( Solid Indigo, #6366f1 )
             
          HomeHero
    ( Outfit Font Wordmark )
```

### 2.2 Secondary Logo (Horizontal Lockup)
Used for desktop headers, navigation bars, and invoices where vertical height is restricted.

```
  [ Shield Symbol ]   HomeHero
```

---

## 3. Spacing, Sizing, & Variations

### 3.1 Clear Space Requirements
To maintain visual impact, keep a minimum clear space around the logo equivalent to the height of the shield icon (`X`):

```
┌──────────────────────────────────────────────┐
│                    Space (X)                 │
│    ┌────────────────────────────────────┐    │
│    │          [Shield (Height: X)]      │    │
│    │             HomeHero               │    │
│    └────────────────────────────────────┘    │
│                    Space (X)                 │
└──────────────────────────────────────────────┘
```

### 3.2 Minimum Dimensions
- **Digital Screens**: Minimum width of **120px** (horizontal lockup) or **80px** (vertical lockup).
- **Print Media**: Minimum width of **30mm** (horizontal) or **20mm** (vertical).

### 3.3 Color Variations
- **Primary**: Brand Indigo (`#6366f1`) shield with a White (`#ffffff`) wordmark on dark backgrounds.
- **Monochrome**: Dark Slate (`#1e293b`) shield and wordmark on light backgrounds.
- **Black & White**: Pure Black (`#000000`) on white packaging, or Pure White (`#ffffff`) on dark backdrops.

---

## 4. Mobile App Icon & Favicon Design

### 4.1 iOS & Android App Icon
- **Design**: The Brand Indigo shield centered on a Slate-950 (`#0b0f19`) square background.
- **Android Adaptive Icon Guidelines**: Keep all active shield graphics within the **66% safe area** circle to prevent clipping during system mask transformations.
- **Export Grid**:
  - iOS App Icon: `1024 x 1024 px` (App Store submission template).
  - Android launcher icons: `512 x 512 px` (Play Store template).

### 4.2 Web Favicons
- **Formats**: SVG format for modern browser compatibility, backed by a legacy `favicon.ico` fallback.
- **Standard Sizes**: `16 x 16 px`, `32 x 32 px`, and `48 x 48 px`.

---

## 5. Logo Usage Policy (DOs & DONTs)

- **DON'T** stretch, skew, or deform the logo shield or wordmark.
- **DON'T** rotate the shield icon independently from the wordmark.
- **DON'T** apply drop shadows, glowing effects, or gradients to the symbol.
- **DO** use the designated solid White logo on dark brand colors (`#0b0f19`).

---

## 6. Export Recommendations & Formats

*   **Vector (.SVG)**: Default format for all web pages and app designs to ensure clean scaling.
*   **Raster (.PNG)**: Exported with transparent backdrops at `300 DPI` for print.
*   **WebP**: Compressed raster format used in emails and web articles to optimize load times.
