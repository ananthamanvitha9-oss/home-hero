# HomeHero - Hyperlocal Home Services Platform (India)

Welcome to the **HomeHero** workspace. This repository contains the complete documentation, database designs, API specifications, and product requirements for HomeHero, a premium on-demand marketplace designed for the Indian home services market.

---

## 🛠️ Technology Stack & Tech Details
*   **Database:** PostgreSQL (with PostGIS extensions for geofencing)
*   **Payments:** Stripe Connect (supports split payouts and escrow holds)
*   **Core UI System Theme:** Royal Indigo (`hsl(245, 75%, 45%)`), Hero Violet (`hsl(270, 80%, 60%)`), and Verified Mint (`hsl(155, 60%, 45%)`)
